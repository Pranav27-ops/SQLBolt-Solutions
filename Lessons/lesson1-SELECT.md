# Lesson 1: SELECT Queries

## Problem Description
Learn the basics of SELECT queries - how to retrieve data from a database table.

## Question
Write a SELECT query to retrieve all columns from a database table.

## Solution
```sql
SELECT * FROM movies;
```

## Key Concepts
- `SELECT` - Used to retrieve data from a database
- `*` - Selects all columns
- `FROM` - Specifies which table to query

## Notes
- The asterisk (*) is a wildcard that means "all columns"
- Always use SELECT when you want to retrieve data
- You can also select specific columns by naming them

## Practice Tips
- Try selecting specific columns instead of using *
- Experiment with different tables
